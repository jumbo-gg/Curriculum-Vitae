#### 概要(レアジョブアプリ iOS/Androidの立ち上げ・開発)

転職の理由として、実家の自営業をしている業務の理解と整理をするために1,2ヶ月休みたかったのと企業スケールの小さいところでアプリ事業を創りたかった。
その中で、自身も過去ユーザーであったレアジョブに縁があり入社。フルスクラッチでiOS/AndroidをRx/MVVMで実装。
途中からはデザイン・マーケ・グロースも担当するようになり無料体験受講率・有料化率をKPIツリーベースで改善。
またSkypeへの依存度の高いビジネルモデルに危機感を感じWebRTCを使った仕組みを検証としてリリース、PMに就任。
フィリピンのエンジニアと協力しつつ、現地にも行きながら開発・ローンチと効果検証まで。

ユーザーがスケールし、1人でやるのに限界を感じていたところCTOが来て面白そうだったのでフロントエンドチームの立ち上げとリーダーに就任。
チームを作りつつ、成果保証型サービスのフロントエンドの設計（Vuex）をリードしつつ、独自レッスンシステムのSPAのパフォーマンス改善を実施。
この時期フィリピンからの出向スタッフが部下になり英語でのマネジメントを経験。

チームがスケールし、技術本部の副部長に就任。また全体最適化を考えてデザインチームのリーダーも就任。
ピープルマネジメントにフォーカスして1on1やコーチング、プロセス改善を数多く実施。現在に至る。
